Additional Design Work

Feel free to contact me for more information on any of the following projects


TCGPlayer AI Landing Page

This design project was my first attempt to use AI-generated images as a design inspiration tool. Much like the classic "Crazy-Eights" UX exercise, AI can be used to rapidly create multiple designs which can then be fleshed out further.

TeamFightTactics Planner Redesign

A quick design project focused on redesigning the planning page for the mobile version of Riot's TeamFight Tactics game The changes made were aimed to create a more robust, detailed plan for once the player is in the game.

ExplorAR Course Prototype

A project created during my time at DePaul University, obtaining my masters degree. ExplorAR was conceptualized as a AR-focused urban exploration mobile application which gave users curated destinations where they could then use the AR-functionality to explore the site. See a detailed report Here.


Nintendo Online Tablet Application Redesign

This redesigned Nintendo Online tablet application was a quick project aimed at tailoring the experience for tablet's instead of the then mirroring of their mobile phone app, using the larger screen space to lessen the travel between multiple different pages.

Anilist Mobile Application

This prototype was designed for a graduate school project and presented various functions and pages of a potential mobile version of Anilist, a web-first anime tracker website.

Food Delivery Manager Dashboard

An original design for a potential web-based dashboard for managers of food delivery companies. The main goal of with this design was to include and display the most relevant and important information on the main homepage, lessening the need for the manager to switch between pages in order to streamline their responsibilities.
